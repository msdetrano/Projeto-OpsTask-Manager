# OpsTask Manager - Gerenciador de Tarefas No-Code

## Sobre o Projeto
Este projeto é um MVP (Produto Mínimo Viável) de um sistema de gerenciamento de tarefas desenvolvido para a disciplina de Fundamentos de No-Code e Low-Code da UniFECAF. O objetivo é validar uma solução de organização interna para times de operações em até 30 dias.

## Tecnologias Utilizadas
* **Plataforma:** Bubble.io (Full-stack No-code)
* **Banco de Dados:** Bubble Built-in Database
* **Frontend:** Bubble Visual Editor (Responsivo)

## Funcionalidades
* **Autenticação:** Login e Cadastro de usuários.
* **Dashboard:** Visualização de tarefas por status (Kanban simplificado).
* **CRUD de Tarefas:** Criação, leitura e atualização de status de tarefas.
* **Colaboração:** Sistema de comentários em tempo real dentro de cada tarefa.
